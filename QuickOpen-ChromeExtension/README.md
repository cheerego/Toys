# ChromeExtensionOpenAll  
一个可以一键打开多种页面的Chrome扩展  
由于经常网上找一些一键打开的扩展，如豆瓣一键打开，微博一键打开等等，然后Chrome商店被强，所以一直都有一种自己写一个的冲动，所以自己就写了一个。也算是锻炼了。

用HTML+JavaScript,数据存储使用HTML5的LocalStorage编写而成。  
虽然丑，但是难看啊。  
> 使用方法
  > * 下载或者git clone
  > * 然后打开Chrome的扩展管理界面
  > * 打开开发者调试模式
  > * 将openall.crx拖入到这个界面中,完成安装
